# ConTinG

Sistema desarrollado en .NET C#, en conjunto con Python. Con una base de datos MySQL, un lector de tarjetas magnéticas, arduino y Raspberry PI. El sistema tiene como propósito gestionar la administración de un centro de estudios. Tomando los accesos de estudiantes, profesores y demás personas pertenecientes al lugar. La cerradura automática de cada puerta solamente podrá ser abierta si un profesor abre la puerta, y los estudiantes solamente podrán ser registrados si la puerta está previamente abierta.
